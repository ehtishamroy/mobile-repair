# Satoshi Variable Font

This directory should contain the Satoshi Variable font files:

- `Satoshi-Variable.woff2` - Modern web font format (preferred)
- `Satoshi-Variable.woff` - Fallback web font format

## How to get the font files:

1. Download Satoshi Variable font from the official source
2. Convert to web formats (woff2, woff) if needed
3. Place the files in this directory

## Usage:

Add the `font-satoshi` class to any element to use Satoshi Variable font:

```html
<h1 class="font-satoshi">Your heading text</h1>
<p class="font-satoshi">Your paragraph text</p>
```

## Fallback:

If Satoshi Variable is not available, the font will fallback to Inter (similar design) or system sans-serif fonts.
